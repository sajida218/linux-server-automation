# Linux Server Automation using Bash

This project automates the installation and configuration of the Nginx web server on an Ubuntu Linux server.

## Features

- Updates system packages
- Installs Nginx
- Starts the Nginx service
- Enables Nginx at boot
- Creates a sample HTML webpage
- Verifies successful deployment

## Requirements

- Ubuntu Server
- sudo privileges

## Run the Script

Make the script executable:

```bash
chmod +x install_nginx.sh
```

Run it:

```bash
./install_nginx.sh
```
